The code folder contains the main scripts for data preprocessing, data partitioning, model definition, training, and evaluation.  All core computational procedures are implemented within this directory. 
The remaining folders (data, model, train_test，importance) are used solely for storing raw data, processed data, intermediate outputs, and generated results.

The datasets used in this project are sourced from:

(1) Imperial College London Big Data Analytical Unit and YouGov Plc., Imperial College London YouGov COVID-19 Behaviour Tracker Data Hub, 2020, Version 1.0 (accessed 1 January 2024).
(2) Hale et al. (2021), A global panel database of pandemic policies (Oxford COVID-19 Government Response Tracker), Nature Human Behaviour, 5:529–538.

The overall methodological framework and analytical pipeline are inspired by:
Ryan et al. (2025), Face mask mandates alter major determinants of adherence to protective health behaviours in Australia, Royal Society Open Science, 12(3):241941.
